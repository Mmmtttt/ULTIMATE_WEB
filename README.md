# 🎯 Ultimate Web - 你的私人漫画与视频管家

> 想象一下，你有一个专属的"私人图书馆"，可以把你在各大平台的收藏全部整合到一起，随时随地阅读和观看，再也不用来回切换 App 了！

---

## 📖 这个项目是做什么的？

### 一句话概括

**Ultimate Web 是一个"内容聚合管理平台"** —— 它可以把你在 JM（禁漫天堂）、PK（哔咔漫画）、JavDB 等平台的收藏夹同步过来，统一管理、统一阅读、统一观看。

### 🌟 核心功能亮点

| 功能 | 说明 |
|------|------|
| 🔄 **收藏夹同步** | 一键同步 JM、PK、JavDB 的收藏内容，告别平台分散 |
| 🔍 **全网搜索** | 在一个地方搜索所有平台的内容，找到就导入 |
| 📚 **在线阅读本子** | 漫画直接在线看，支持左右翻页、上下滚动、缩放、全屏 |
| 🎬 **在线播放视频** | 视频一键播放，支持多清晰度切换 |
| 🏷️ **标签管理** | 给内容打标签，按标签筛选，找东西快到飞起 |
| ⭐ **评分系统** | 1-12 分精细评分，按评分排序，好内容一目了然 |
| 📋 **清单功能** | 创建"我的最爱"、"待看清单"等，分类管理更清晰 |
| 👤 **作者/演员订阅** | 关注喜欢的作者或演员，一键检查更新 |
| 📱 **多端适配** | 电脑、手机都能用，界面自动适配 |

### 🎬 使用场景举例

**场景一：收藏夹大整合**
> 你在 JM 收藏了 200 本本子，在 PK 收藏了 100 本，在 JavDB 收藏了 50 部视频。以前要看什么得打开三个 App，现在全部同步到 Ultimate Web，一个页面全搞定！

**场景二：高效找内容**
> 想看"画工好"的"纯爱"本子？点击标签筛选，两秒出结果。想找某个作者的所有作品？点作者名直接跳转。想看自己打过高分的？按评分排序即可。

**场景三：追更不遗漏**
> 关注了几个喜欢的作者，想看他们有没有更新？点"检查更新"，一键扫描所有订阅作者的新作品。

## 🚀 怎么使用？

### 第一步：获取项目

有两种方式可以获取项目，选择适合你的方式：

---

#### 方式一：下载打包好的程序（推荐新手）

**最简单的方式！** 直接下载我已经打包好的程序，无需安装任何依赖。

1. 打开项目主页：[https://github.com/Mmmtttt/ULTIMATE_WEB](https://github.com/Mmmtttt/ULTIMATE_WEB)
2. 点击右侧的 **Releases** 栏
3. 下载对应平台的程序包：
   - **Windows**：`UltimateWeb_Windows_x.x.x.zip`
   - **Linux**：`UltimateWeb_Linux_x.x.x.tar.gz`
   - **Android**：`UltimateWeb_Android_x.x.x.apk`
4. 解压后直接运行即可

**优点**：
- ✅ 无需安装 Python、Node.js 等环境
- ✅ 解压即用，开箱即用
- ✅ 适合不想折腾的用户

---

#### 方式二：克隆源码并安装依赖（适合开发者）

如果你想自己编译或参与开发，可以克隆源码：

1. **克隆仓库（记得带上子库）**
   ```bash
   # 克隆仓库并同步子模块
   git clone --recursive https://github.com/Mmmtttt/ULTIMATE_WEB.git

   # 如果已经克隆但忘记带子模块，可以执行：
   git submodule update --init --recursive
   ```

2. **安装后端依赖**
   ```bash
   cd ULTIMATE_WEB/comic_backend
   pip install -r requirements.txt
   ```

3. **安装前端依赖**
   ```bash
   cd ../comic_frontend
   npm install
   ```

**优点**：
- ✅ 可以获取最新代码
- ✅ 可以自定义修改
- ✅ 可以参与项目开发

---

### 第二步：选择工作模式

Ultimate Web 支持三种工作模式，你可以根据自己的情况选择：

---

### 模式一：Windows 内网服务器（推荐新手）

**适合人群**：有一台 Windows 电脑，想在家里用手机/平板/电脑访问

**工作原理**：
```
你的电脑运行 Ultimate Web 服务
        ↓
手机/平板/电脑通过浏览器访问（同一 WiFi 下）
```

**使用步骤**：

**如果你下载了打包程序**：
1. 解压 `UltimateWeb_Windows_x.x.x.zip`
2. 双击运行 `start.exe`
3. 浏览器访问 `http://localhost:5173`

**如果你克隆了源码**：
```powershell
# 回到项目根目录，运行启动脚本
.\start_project.ps1
```

**访问系统**：
- 电脑访问：打开浏览器，输入 `http://localhost:5173`
- 手机访问：确保手机和电脑连同一个 WiFi，在手机浏览器输入 `http://你的电脑IP:5173`

**优点**：
- ✅ 设置简单，跟着步骤走就行
- ✅ 手机、平板、电脑都能访问
- ✅ 数据存在自己电脑上，安全私密
- ✅ **功能最完整**：支持全网搜索、第三方同步、视频代理等全部能力

**缺点**：
- ❌ 电脑要一直开着
- ❌ 只能在同一 WiFi 下访问

---

### 模式二：Linux 公网服务器

**适合人群**：有自己的云服务器（如阿里云、腾讯云），想随时随地访问

**工作原理**：
```
云服务器运行 Ultimate Web 服务
        ↓
任何设备通过公网 IP 访问（有网就能用）
```

**使用步骤**：

**如果你下载了打包程序**：
1. 上传 `UltimateWeb_Linux_x.x.x.tar.gz` 到服务器
2. 解压：`tar -xzf UltimateWeb_Linux_x.x.x.tar.gz`
3. 运行：`./start`

**如果你克隆了源码**：
```bash
# 上传项目到服务器
scp -r ULTIMATE_WEB user@your-server:/home/user/

# 安装依赖并启动
cd /home/user/ULTIMATE_WEB/comic_backend
pip3 install -r requirements.txt

cd ../comic_frontend
npm install
npm run build

# 启动服务
nohup python3 comic_backend/app.py &
```

**配置 Nginx 反向代理（可选）**：
- 配置域名访问，更方便记忆

**优点**：
- ✅ 随时随地访问，不受网络限制
- ✅ 服务器稳定，24 小时运行
- ✅ **功能最完整**：支持全网搜索、第三方同步、视频代理等全部能力

**缺点**：
- ❌ 需要购买服务器（有成本）
- ❌ 配置相对复杂

---

### 模式三：安卓 App（移动端专属）

**适合人群**：主要用手机使用，没有电脑或服务器

**工作原理**：
```
手机 App 内置轻量后端
        ↓
通过网络连接 Windows/Linux 服务器同步数据
```

**使用步骤**：

1. **下载 APK 安装包**
   - 从 [GitHub Release](https://github.com/Mmmtttt/ULTIMATE_WEB/releases) 下载安卓安装包

2. **安装并打开**
   - 安装后打开 App 即可使用

3. **配置同步服务器（可选）**
   - 如果你有 Windows/Linux 服务器，可以配置服务器地址
   - App 会自动同步服务器上的数据到本地

**优点**：
- ✅ 最简单的使用方式
- ✅ 不需要电脑或服务器也能用
- ✅ 支持离线阅读已缓存的内容

**缺点**：
- ❌ 只能在手机上用
- ❌ **功能有裁剪**（详见下方说明）

#### ⚠️ 安卓端功能说明

安卓端采用"轻客户端 + 同步中台"架构，部分功能有所裁剪：

| 功能 | Windows/Linux | 安卓端 | 说明 |
|------|:-------------:|:------:|------|
| 本地库管理 | ✅ | ✅ | 查看、筛选、排序已导入内容 |
| 预览库浏览 | ✅ | ✅ | 浏览推荐内容 |
| 在线阅读漫画 | ✅ | ✅ | 支持翻页、缩放、进度保存 |
| 在线播放视频 | ✅ | ✅ | 支持多清晰度切换 |
| 标签管理 | ✅ | ✅ | 创建、编辑、删除标签 |
| 清单管理 | ✅ | ✅ | 创建、编辑清单 |
| 订阅管理 | ✅ | ✅ | 关注作者/演员 |
| 本地搜索 | ✅ | ✅ | 在已导入内容中搜索 |
| **全网搜索** | ✅ | ❌ | 安卓端不支持直接搜索第三方平台 |
| **第三方同步** | ✅ | ❌ | 安卓端不能直接同步 JM/PK/JavDB |
| **视频代理** | ✅ | ❌ | 安卓端不支持视频代理功能 |

**💡 安卓端最佳实践**：

如果你想在安卓端使用全网搜索和第三方同步功能，建议这样搭配：

```
┌─────────────────────────────────────────────────────┐
│  Windows/Linux 服务器（完整功能）                    │
│  - 同步 JM/PK/JavDB 收藏夹                          │
│  - 全网搜索并导入内容                                │
│  - 视频代理播放                                      │
└─────────────────────┬───────────────────────────────┘
                      │ 数据同步
                      ▼
┌─────────────────────────────────────────────────────┐
│  安卓 App（轻量客户端）                              │
│  - 同步服务器数据到本地                              │
```

### 10. 数据同步功能（安卓端核心）

安卓端支持与 Windows/Linux 服务器进行数据同步：

**同步内容**：
- 📚 漫画/视频元数据（标题、作者、标签、评分等）
- 📖 阅读进度
- 🏷️ 标签和清单
- 👤 订阅的作者/演员
- ⚙️ 用户配置

**同步方式**：
- **增量同步**：只同步变更的数据，节省流量
- **双向同步**：本地修改会同步到服务器，服务器更新也会同步到本地
- **离线支持**：无网络时可离线阅读已缓存内容，网络恢复后自动同步

**配置方法**：
1. 在安卓端进入「系统设置」
2. 点击「同步服务器配置」
3. 输入服务器地址（如 `http://192.168.1.100:5000`）
4. 点击「测试连接」确认可用
5. 点击「立即同步」开始同步

## 🔧 平台账号配置

要使用同步和搜索功能，需要配置平台账号（仅 Windows/Linux）：

### 配置文件位置

```
comic_backend/third_party_config.json
```

### 配置示例

```json
{
  "adapters": {
    "jmcomic": {
      "username": "你的JM用户名",
      "password": "你的JM密码"
    },
    "picacomic": {
      "account": "你的PK邮箱",
      "password": "你的PK密码"
    },
    "javdb": {
      "cookies": {}
    }
  }
}
```

### 配置说明

| 平台 | 必填项 | 说明 |
|------|--------|------|
| JM（禁漫天堂） | 用户名、密码 | 用于同步收藏夹 |
| PK（哔咔漫画） | 邮箱、密码 | 用于同步收藏夹和搜索 |

## ❓ 常见问题

### Q1：启动失败怎么办？

**检查以下几点**：
1. 是否安装了 Python 3.8+ 和 Node.js 16+
2. 端口 5000 和 5173 是否被占用
3. 是否正确安装了依赖

### Q2：手机无法访问电脑上的服务？

**解决方法**：
1. 确保手机和电脑连同一个 WiFi
2. 检查电脑防火墙是否放行端口
3. 使用电脑的局域网 IP（不是 localhost）

### Q3：同步收藏夹失败？

**可能原因**：
1. 账号密码配置错误
2. 平台服务器暂时不可用
3. 网络问题

### Q4：数据会丢失吗？

**不会！** 系统有三级自动备份：
- 每 10 分钟备份一次
- 每小时备份一次
- 每天备份一次

即使数据损坏，也可以从备份恢复。

### Q5：安卓端如何同步服务器数据？

1. 确保 Windows/Linux 服务器正在运行
2. 确保手机能访问服务器（同一 WiFi 或公网）
3. 在安卓端配置服务器地址
4. 点击「立即同步」

### Q6：安卓端为什么没有全网搜索？

安卓端采用轻量化设计，不包含第三方平台采集能力。如需使用全网搜索，请：
1. 在 Windows/Linux 服务器上搜索并导入内容
2. 安卓端同步服务器数据后即可查看
