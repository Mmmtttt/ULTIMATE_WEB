# 20260227_233000_006 - 懒加载和图片路径问题修复

## 问题描述

1. **懒加载指令报错**：`Failed to resolve directive: lazy`
2. **详情页缩略图不显示**：图片路径使用了相对路径 `/api/v1/...`
3. **详情页标签显示不正确**：显示的是 `tag_xxx` 而不是标签名称
4. **阅读页图片不显示**：同样使用了相对路径

## 问题原因

### 1. 懒加载指令未注册
Vant 的 `lazy-load` 指令需要单独注册，全局注册 Vant 时不会自动包含 Lazyload 插件。

### 2. 图片路径问题
前端使用了相对路径 `/api/v1/comic/image?...`，但应该使用完整 URL `http://127.0.0.1:5000/api/v1/...`

### 3. 标签显示问题
详情页只获取了漫画数据，没有获取标签列表，无法将 `tag_id` 转换为标签名称。

## 修复方案

### 1. 注册 Lazyload 插件
修改 [main.js](file:///d:/code/ULTIMATE_WEB/comic_frontend/src/main.js)：
```javascript
import Vant, { Lazyload } from 'vant'
app.use(Lazyload)
```

### 2. 修复图片路径
修改 [ComicDetail.vue](file:///d:/code/ULTIMATE_WEB/comic_frontend/src/views/ComicDetail.vue)：
```javascript
// 获取图片URL
const getImageUrl = (comicId, pageNum) => {
  return `http://127.0.0.1:5000/api/v1/comic/image?comic_id=${comicId}&page_num=${pageNum}`
}
```

修改 [ComicReader.vue](file:///d:/code/ULTIMATE_WEB/comic_frontend/src/views/ComicReader.vue)：
```javascript
// 转换图片路径为完整URL
images.value = response.data.map((path, index) => getImageUrl(comicId.value, index + 1))
```

### 3. 修复标签显示
修改 [ComicDetail.vue](file:///d:/code/ULTIMATE_WEB/comic_frontend/src/views/ComicDetail.vue)：
```javascript
// 获取所有标签
const response = await fetch('http://127.0.0.1:5000/api/v1/tags')
allTags.value = result.data

// 获取标签名称
const getTagName = (tagId) => {
  const tag = allTags.value.find(t => t.id === tagId)
  return tag ? tag.name : tagId
}
```

### 跨域图片处理
