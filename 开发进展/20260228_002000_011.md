# 20260228_002000_011 - 阅读页面滑动优化

## 问题描述

1. **滑动迟滞阻塞感**：上下滑动时不够流畅，有卡顿感
2. **图片之间有缝隙**：缩小页面比例后，图片之间仍有间隙

## 问题原因

### 1. 滑动迟滞
- 没有启用 GPU 加速
- 缺少性能优化属性
- 图片加载阻塞滚动

### 2. 图片间隙
- `flex: 0 0 100vw` 使用 `vw` 单位可能导致小数像素间隙
- 图片默认是 `inline` 元素，有默认间距
- 缺少 `margin: 0` 和 `padding: 0` 重置

## 修复方案

### 1. 优化滑动性能
- 添加 `will-change: scroll-position` 启用 GPU 加速
- 添加 `transform: translateZ(0)` 强制硬件加速
- 添加 `backface-visibility: hidden` 优化渲染
- 添加 `loading="lazy"` 和 `decoding="async"` 优化图片加载

### 2. 消除图片间隙
- 使用 `flex: 0 0 100%` 代替 `flex: 0 0 100vw`
- 添加 `display: block` 消除图片默认间距
- 添加 `vertical-align: top` 防止基线对齐问题
- 添加 `margin: 0` 和 `padding: 0` 重置
- 添加 `box-sizing: border-box` 统一盒模型

```css

.left-right-mode {
  /* 优化性能 */
  will-change: scroll-position;

```

### 消除间隙
```css
.page {
  flex: 0 0 100%;  /* 使用 % 代替 vw */
  width: 100%;
  /* 消除间隙 */
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

.comic-image {
  width: 100%;
  height: 100%;
  /* 消除图片间隙 */
  display: block;
  vertical-align: top;
}

.up-down-page {
  width: 100%;
  /* 防止margin合并 */
  overflow: hidden;
  /* 消除间隙 */
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```

## 技术说明

### GPU 加速原理
- `will-change: scroll-position` - 提示浏览器优化滚动性能
- `transform: translateZ(0)` - 强制创建 GPU 图层
- `backface-visibility: hidden` - 优化 3D 变换性能

### 消除间隙原理
- `display: block` - 图片默认是 inline 元素，有默认间距
- `vertical-align: top` - 防止基线对齐导致的间隙
- `box-sizing: border-box` - 统一盒模型计算方式
- `overflow: hidden` - 防止 margin 合并

### 图片加载优化
- `loading="lazy"` - 懒加载，进入视口才加载
- `decoding="async"` - 异步解码，不阻塞主线程
