# 软连接导入：阶段3完成记录（2026-03-31）

## 本阶段目标
- 打通 `soft_ref` 漫画在阅读链路中的无感访问能力（目录源 + 压缩包源）。
- 支持加密压缩包首次访问密码挑战，用户提交后可继续阅读。
- 在源文件缺失时返回明确错误，不静默失败。

## 实现内容
- 新增服务：`comic_backend/application/softref_comic_reader.py`
  - 统一读取 `soft_ref` 漫画来源（普通目录、zip/rar/7z、嵌套压缩包）。
  - 建立页索引并按页读取图片流。
  - 支持 archive fingerprint 维度密码存储与复用（明文）。
  - 源路径缺失时抛出可识别异常。
- 漫画实体补齐软连接字段：
  - `import_source`
  - `storage_mode`
  - `soft_ref_locator`
  - 文件：`comic_backend/domain/comic/entity.py`
- 漫画读取 API 适配：
  - `GET /api/v1/comic/images`：软连接漫画分流读取；加密场景返回 `password_required` 挑战对象。
  - `GET /api/v1/comic/image`：软连接漫画按页流式返回；未提供密码时返回业务错误（code=428）。
  - `POST /api/v1/comic/softref/password`：保存压缩包密码。
  - 文件：`comic_backend/api/v1/comic.py`
- 前端阅读页适配：
  - `comicApi.setSoftRefPassword(...)`
  - `comicStore.fetchImages(...)` 兼容 `images + password_required` 载荷。
  - 阅读页在密码挑战时弹窗输入密码并自动重试。
  - 文件：
    - `comic_frontend/src/api/comic.js`
    - `comic_frontend/src/stores/comic.js`
    - `comic_frontend/src/views/ComicReader.vue`
