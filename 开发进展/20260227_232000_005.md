# 20260227_232000_005 - Vant UI 导入问题修复

## 问题描述

浏览器控制台报错：
```
SyntaxError: The requested module '/node_modules/.vite/deps/vant.js?v=64dec752' 
does not provide an export named 'VanButton'
```

## 问题原因

Vant UI 已经在 `main.js` 中全局注册：
```javascript
import Vant from 'vant'
app.use(Vant)
```

但在各个视图组件中又尝试单独导入 Vant 组件：
```javascript
import { VanNavBar, VanImage, VanEmpty, VanButton, ... } from 'vant'
```

这导致了导入冲突和错误。

## 修复方案

移除所有视图组件中的 Vant 组件导入语句，因为 Vant 已经全局注册，可以直接在模板中使用。

## 技术说明

### Vant UI 的两种使用方式

1. **全局注册**（本项目使用的方式）
```javascript
// main.js
import Vant from 'vant'
app.use(Vant)

// 组件中直接使用，无需导入
<template>
  <van-button>按钮</van-button>
</template>
```

2. **按需导入**（另一种方式）
```javascript
// 组件中单独导入
import { Button } from 'vant'

<template>
  <van-button>按钮</van-button>
</template>
```

### 注意事项

- 全局注册方式下，不要在组件中重复导入 Vant 组件
- 按需导入需要配置 babel-plugin-import 插件
- 全局注册会增加打包体积，但使用更方便
