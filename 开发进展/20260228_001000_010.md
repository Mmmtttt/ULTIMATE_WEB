# 20260228_001000_010 - 后端服务崩溃修复

## 问题描述

网页打不开，浏览器报错：
```
Failed to load resource: the server responded with a status of 502 (Bad Gateway)
net::ERR_CONNECTION_REFUSED
```

## 问题原因

后端服务崩溃，Python 进程已停止。

重启服务时报错：
```
ImportError: cannot import name 'json_handler' from 'utils.json_handler'
```

问题原因：
1. **utils/__init__.py 为空**：没有导出 `json_handler` 实例
2. **导入路径错误**：`comic.py` 从 `utils.json_handler` 导入，但应该通过 `utils` 包导入

## 修复方案

### 1. 修复 utils/__init__.py
添加导出语句，创建全局 `json_handler` 实例：
```python
from .json_handler import JsonHandler

# 创建全局 json_handler 实例
json_handler = JsonHandler()

__all__ = ['JsonHandler', 'json_handler']
```

### 2. 修复 comic.py 导入路径
修改导入语句：
```python
# 修改前
from utils.json_handler import json_handler

# 修改后
from utils import json_handler
```

## 技术说明

### Python 包导入机制
1. `from utils.json_handler import json_handler` - 尝试从模块导入变量
2. `from utils import json_handler` - 从包的 `__init__.py` 导入

### 全局实例模式
在 `__init__.py` 中创建全局实例，方便其他模块直接使用：
```python
# utils/__init__.py
json_handler = JsonHandler()

# 其他模块使用
from utils import json_handler
```
