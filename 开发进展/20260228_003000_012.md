# 20260228_003000_012 - 图片缩放功能

## 问题描述

用户需要查看图片细节，但图片无法放大。

## 解决方案

添加完整的图片缩放功能，支持：
1. **按钮缩放**：点击 +/- 按钮放大/缩小
2. **滚轮缩放**：鼠标滚轮快速缩放
3. **拖拽平移**：放大后可以拖拽查看不同区域
4. **双指缩放**：移动端双指捏合缩放
5. **双击还原**：单击放大后的图片还原

## 修复的文件

- ✅ [ComicReader.vue](file:///d:/code/ULTIMATE_WEB/comic_frontend/src/views/ComicReader.vue)

## 功能说明

### 缩放控制
- 点击底部"缩放"按钮显示/隐藏缩放控制栏
- 缩放控制栏显示在右侧中间位置
- 显示当前缩放比例（100% - 500%）

### 操作方式

#### 桌面端
1. **滚轮缩放**：在图片上滚动鼠标滚轮
   - 向上滚动：放大
   - 向下滚动：缩小
2. **按钮缩放**：点击右侧 +/- 按钮
3. **拖拽平移**：放大后按住鼠标拖拽
4. **单击还原**：点击图片还原到 100%

#### 移动端
1. **双指缩放**：双指捏合/张开
2. **单指拖拽**：放大后单指拖拽
3. **按钮缩放**：点击 +/- 按钮

### 缩放范围
- 最小：100%（原始大小）
- 最大：500%（5倍放大）
- 步进：50%（每次增减 50%）

## 关键代码

### 缩放状态管理
```javascript
const zoomLevel = ref(1)           // 当前缩放级别
const showZoomControls = ref(false) // 是否显示缩放控制栏
const isDragging = ref(false)       // 是否正在拖拽
const dragOffset = ref({ x: 0, y: 0 }) // 拖拽偏移量
```

### 缩放样式计算
```javascript
const getZoomStyle = () => {
  return {
    transform: `scale(${zoomLevel.value}) translate(${dragOffset.value.x}px, ${dragOffset.value.y}px)`,
    cursor: zoomLevel.value > 1 ? (isDragging.value ? 'grabbing' : 'grab') : 'pointer'
  }
}
```

### 滚轮缩放
```javascript
const handleWheel = (e) => {
  e.preventDefault()
  if (e.deltaY < 0) {
    zoomIn()
  } else {
    zoomOut()
  }
}
```

### 鼠标拖拽
```javascript
const handleMouseDown = (e) => {
  if (zoomLevel.value > 1) {
    isDragging.value = true
    dragStart.value = { x: e.clientX - dragOffset.value.x, y: e.clientY - dragOffset.value.y }
  }
}

const handleMouseMove = (e) => {
  if (isDragging.value && zoomLevel.value > 1) {
    dragOffset.value = {
      x: e.clientX - dragStart.value.x,
      y: e.clientY - dragStart.value.y
    }
  }
}
```

### 双指缩放（移动端）
```javascript
const handleTouchStart = (e) => {
  if (e.touches.length === 2) {
    lastTouchDistance.value = getTouchDistance(e.touches)
  }
}

const handleTouchMove = (e) => {
  if (e.touches.length === 2) {
    const distance = getTouchDistance(e.touches)
    const scale = distance / lastTouchDistance.value
    zoomLevel.value = Math.max(1, Math.min(5, zoomLevel.value * scale))
    lastTouchDistance.value = distance
  }
}
```

## 技术说明

### 性能优化
- 使用 `transform` 实现缩放，启用 GPU 加速
- 缩放时禁用 `scroll-snap`，避免冲突
- 添加 `transition` 实现平滑动画

### 交互设计
- 放大后鼠标变为 `grab`/`grabbing` 指示可拖拽
- 拖拽时禁用页面滚动
- 切换页面时自动重置缩放
