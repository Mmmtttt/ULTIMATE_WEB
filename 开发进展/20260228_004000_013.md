# 20260228_004000_013 - 桌面端交互逻辑优化

## 问题描述

桌面端交互逻辑需要调整：
1. 滚轮应该用于翻页而不是缩放
2. Ctrl + 滚轮用于缩放
3. 鼠标拖拽用于翻页

## 解决方案

区分桌面端和移动端的不同交互逻辑：

### 桌面端（鼠标操作）
- **滚轮**：翻页（上下滚动）
- **Ctrl + 滚轮**：缩放
- **鼠标拖拽**：翻页（左右/上下拖拽）
- **放大后拖拽**：平移查看细节

### 移动端（触摸操作）
- **双指捏合**：缩放
- **单指拖拽**：平移（放大后）
- **滑动**：翻页

### 滚轮处理（桌面端）
```javascript
const handleWheel = (e) => {
  // 触摸设备使用原有逻辑
  if (isTouchDevice()) {
    // ... 移动端逻辑
    return
  }

  // 桌面端逻辑
  if (e.ctrlKey) {
    // Ctrl + 滚轮 = 缩放
    e.preventDefault()
    if (e.deltaY < 0) zoomIn()
    else zoomOut()
  } else {
    // 普通滚轮 = 翻页
    e.preventDefault()
    // 根据模式进行翻页
    if (pageMode.value === 'left_right') {
      // 左右模式：滚轮控制水平滚动
      if (e.deltaY > 0) {
        container.scrollBy({ left: window.innerWidth, behavior: 'smooth' })
      } else {
        container.scrollBy({ left: -window.innerWidth, behavior: 'smooth' })
      }
    } else {
      // 上下模式：滚轮控制垂直滚动
      if (e.deltaY > 0) {
        container.scrollBy({ top: window.innerHeight - 46, behavior: 'smooth' })
      } else {
        container.scrollBy({ top: -(window.innerHeight - 46), behavior: 'smooth' })
      }
    }
  }
}
```

### 鼠标拖拽（桌面端）
```javascript
const handleMouseDown = (e) => {
  if (isTouchDevice()) return // 触摸设备使用原有逻辑

  if (zoomLevel.value > 1) {
    // 放大状态下拖拽平移
    isDragging.value = true
    dragStart.value = { x: e.clientX - dragOffset.value.x, y: e.clientY - dragOffset.value.y }
  } else {
    // 正常状态下拖拽翻页
    isDragging.value = true
    dragStart.value = { x: e.clientX, y: e.clientY }
  }
}

const handleMouseUp = (e) => {
  if (isTouchDevice()) return // 触摸设备使用原有逻辑

  if (zoomLevel.value === 1) {
    // 正常状态下计算拖拽距离翻页
    const deltaX = e.clientX - dragStart.value.x
    const deltaY = e.clientY - dragStart.value.y
    const threshold = 50 // 翻页阈值

    if (pageMode.value === 'left_right') {
      // 左右模式：水平拖拽
      if (deltaX < -threshold) nextPage()      // 向左拖拽 = 下一页
      else if (deltaX > threshold) prevPage()  // 向右拖拽 = 上一页
    } else {
      // 上下模式：垂直拖拽
      if (deltaY < -threshold) nextPage()      // 向上拖拽 = 下一页
      else if (deltaY > threshold) prevPage()  // 向下拖拽 = 上一页
    }
  }

  isDragging.value = false
}
```

### 触摸事件（移动端）
```javascript
const handleTouchStart = (e) => {
  if (!isTouchDevice()) return // 只在触摸设备上处理

  if (e.touches.length === 2) {
    // 双指缩放开始
    lastTouchDistance.value = getTouchDistance(e.touches)
  } else if (e.touches.length === 1 && zoomLevel.value > 1) {
    // 单指拖拽开始
    isDragging.value = true
    dragStart.value = { 
      x: e.touches[0].clientX - dragOffset.value.x, 
      y: e.touches[0].clientY - dragOffset.value.y 
    }
  }
}

const handleTouchMove = (e) => {
  if (!isTouchDevice()) return

  if (e.touches.length === 2) {
    // 双指缩放
    e.preventDefault()
    const distance = getTouchDistance(e.touches)
    const scale = distance / lastTouchDistance.value
    zoomLevel.value = Math.max(1, Math.min(5, zoomLevel.value * scale))
    lastTouchDistance.value = distance
  } else if (e.touches.length === 1 && isDragging.value && zoomLevel.value > 1) {
    // 单指拖拽
    e.preventDefault()
    dragOffset.value = {
      x: e.touches[0].clientX - dragStart.value.x,
      y: e.touches[0].clientY - dragStart.value.y
    }
  }
}
```

## 交互逻辑对比

| 操作 | 桌面端（鼠标） | 移动端（触摸） |
|------|---------------|---------------|
| 滚轮/滑动 | 翻页 | 翻页 |
| Ctrl + 滚轮 | 缩放 | - |
| 双指捏合 | - | 缩放 |
| 拖拽（正常） | 翻页 | 翻页 |
| 拖拽（放大后） | 平移 | 平移 |
| 单击 | 显示菜单/还原 | 显示菜单/还原 |
