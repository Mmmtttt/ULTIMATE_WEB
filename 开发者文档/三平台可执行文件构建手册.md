## 2. One-click way to build all 3 targets (recommended)

Because Windows and Linux executables are host-OS specific with PyInstaller, the recommended one-click method is CI matrix.

Use workflow:

- `.github/workflows/release-three-platforms.yml`

### 2.1 Full command list

```bash
git clone <your-repo-url>
cd ULTIMATE_WEB
git submodule update --init --recursive

git checkout -b feat/your-change
# edit code...
git add .
git commit -m "feat: your change"
git push origin feat/your-change

gh auth login
gh workflow run release-three-platforms.yml --ref feat/your-change
RUN_ID=$(gh run list --workflow release-three-platforms.yml --branch feat/your-change --limit 1 --json databaseId --jq '.[0].databaseId')
gh run watch "$RUN_ID"

mkdir -p output/from_ci
gh run download "$RUN_ID" -n ultimate-release-bundle -D output/from_ci
```

### 2.2 Final output locations (after download)

- Windows EXE:
  - `output/from_ci/ultimate-windows/dist/ultimate_backend_windows.exe`
- Linux executable:
  - `output/from_ci/ultimate-linux/dist/ultimate_backend_linux`
- Android APK:
  - `output/from_ci/ultimate-android/apk/app-debug.apk`

## 3. Local Windows build for Android (explicit steps)

This section is for building Android APK on Windows directly.

## 3.1 Install Java 17

```powershell
winget install EclipseAdoptium.Temurin.17.JDK --accept-package-agreements --accept-source-agreements
```

Re-open terminal, then verify:

```powershell
java -version
```

## 3.2 Install Android SDK command-line tools

```powershell
$sdkRoot = "$env:LOCALAPPDATA\Android\Sdk"
New-Item -ItemType Directory -Force -Path "$sdkRoot\cmdline-tools" | Out-Null

Invoke-WebRequest -Uri "https://dl.google.com/android/repository/commandlinetools-win-11076708_latest.zip" -OutFile "$env:TEMP\cmdline-tools.zip"
Expand-Archive -Path "$env:TEMP\cmdline-tools.zip" -DestinationPath "$sdkRoot\cmdline-tools" -Force

if (Test-Path "$sdkRoot\cmdline-tools\latest") { Remove-Item -Recurse -Force "$sdkRoot\cmdline-tools\latest" }
Move-Item "$sdkRoot\cmdline-tools\cmdline-tools" "$sdkRoot\cmdline-tools\latest"

$env:ANDROID_SDK_ROOT = $sdkRoot
$env:Path = "$sdkRoot\cmdline-tools\latest\bin;$sdkRoot\platform-tools;$env:Path"

sdkmanager --licenses
sdkmanager "platform-tools" "platforms;android-34" "build-tools;34.0.0"
```

Verify:

```powershell
sdkmanager --version
adb --version
```

## 3.3 Build Android APK from this repo

```powershell
git clone <your-repo-url>
cd ULTIMATE_WEB
git submodule update --init --recursive

python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip

npm ci --prefix comic_frontend

python scripts\release_unified.py --targets android --execute --build-output output\multi_target --package-output output\packages
```

Final APK location:

- `output/packages/android/apk/app-debug.apk`
