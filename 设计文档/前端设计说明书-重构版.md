# ULTIMATE_WEB 前端设计说明书（重构版）

## 1. 文档目标
本说明书基于当前 `comic_frontend` 代码，梳理前端对用户提供的全部功能，并标注核心实现位置。
本次重构遵循：
- 现有功能不变
- 减少重复代码与重复逻辑
- 删除确认未使用的冗余代码

## 4. 关键功能耦合关系

### 4.1 详情页 -> 筛选页
- 在详情页点击作者/标签后，跳转 `Library/Preview` 并附带 query（`author` / `tagId`）
- `Library.vue`、`Preview.vue` 监听 query 自动应用高级筛选

### 4.2 列表/详情 -> 清单系统
- 收藏本质是向默认收藏清单绑定/解绑
- 清单详情页可继续筛选与排序，且可直接移除条目
- 详情页“加入清单”与清单管理页共享 `listStore` 接口

### 4.3 导入链路
- 全网搜索选中结果 -> 导入任务创建（漫画）或直接导入接口（视频）
- “我的”页、创作者详情页、清单导入页均可触发导入
- 导入进度统一在 `ImportTasks` 页面查看

### 4.4 标签链路
- 标签管理页支持批量打标
- 标签详情页支持按来源（本地/推荐）查看并跳详情
- 详情页标签点击可回流到库页筛选

### 4.5 回收站链路
- 列表/详情页可移入回收站
- 回收站提供恢复与永久删除，支持批量
